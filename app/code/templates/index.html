Welcome to unKnown

unKnown is a tool that will help anonymize your dataset using k-anonymity. Click "Get Started" to import your data or "Tutorials" to learn how to use this tool to anonymize your dataset.

About unKnown

unKnown was developed as a course project for 08-605: Engineering Privacy in Software at Carnegie Mellon University. During the development of the tool, the team collaborated with the Western Pennsylvania Data Center.

Tutorials

View our tutorials to learn more about the theory behind k-anonymity, the requirements for running unKnown, and a step-by-step guide on how to anonymize your dataset.
